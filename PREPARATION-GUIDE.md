# 상견례 준비 가이드

## 사전 준비 (상견례 전에 미리 해두세요!)

### 1단계: 방 생성 및 사진 등록

1. **방 만들기**
   - `/host` 페이지에서 "새 방 만들기" 클릭
   - 생성된 방 코드를 **꼭 기록해두세요!** (예: 24591)

2. **사진 등록**
   - 최대 30장의 사진 URL 입력
   - Imgur나 Supabase Storage 사용 권장
   - 모든 사진 등록 완료

3. **라운드 생성**
   - "라운드 생성하기" 버튼 클릭
   - 사진들이 자동으로 3장씩 그룹화됩니다

4. **준비 완료 확인**
   - ✅ "준비 완료! 현장에서 바로 사용할 수 있습니다." 메시지 확인

### 2단계: 방 코드 저장

**중요:** 방 코드를 반드시 기록해두세요!

- 방 코드 복사 버튼 클릭
- 메모장이나 메시지에 저장
- 현장에서 사용할 기기에 미리 저장

## 현장에서 사용하기

### 호스트 (TV/아이패드)

1. `/host` 페이지 접속
2. "기존 방 불러오기"에 방 코드 입력
3. "불러오기" 클릭
4. "호스트 화면 열기" 클릭
5. TV/아이패드에 화면 표시

### 참가자 (가족들)

1. `/join` 페이지 접속
2. 방 코드 입력
3. 가족 선택 (신랑네/신부네/우리부부)
4. 라운드 시작 대기

### 진행 순서

1. 호스트가 "라운드 시작" 버튼 클릭
2. 가족들이 각자 사진 선택
3. 모든 가족이 선택하면 "라운드 종료" 클릭
4. 다음 라운드로 진행
5. 모든 라운드 완료 후 결과 확인

## 팁

- **인터넷 연결 확인**: 모든 기기가 같은 Wi-Fi에 연결되어 있어야 합니다
- **방 코드 공유**: 가족들에게 방 코드를 미리 알려주세요
- **테스트**: 상견례 전에 한 번 테스트해보세요!
